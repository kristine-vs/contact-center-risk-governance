# Contact Center Risk & Governance Analytics

This project simulates and analyzes legal, compliance, and escalation risk in a financial services contact center.

## Overview

This project simulates a governance and escalation monitoring system
for a regulated contact center environment.

It demonstrates:
- Risk detection logic
- Escalation validation
- Agent compliance monitoring
- Audit review workflows
- Failure impact analysis

Data is synthetically generated in Python with embedded governance rules
and failure injection.

Dashboards built in Power BI using DAX.

## Tech Stack
- Python (Pandas, NumPy, Faker)
- Power BI
- DAX
